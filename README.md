# ConeTiming
Calculate time taken to run between two cones using YOLOv5s model for cone detection and pose estimation for whether subject has crossed the cone. Created using mediapipe, opencv, pytorch and web app backend using Flask.

# Quick Run
1. install dependencies using `pip install -r requirements.txt`
2. run python file using `python app.py`
3. Visit localhost to view the app (`http://127.0.0.1:5000/`)
